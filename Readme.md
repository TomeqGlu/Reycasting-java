# Projekt implementujący silnik raycastingu w stylu Wolfensteina 3D napisany w czystej Javie.

**Autor: Tomasz Głuch 16744**

Ręczne uruchamianie:

**Linux:**
`chmod +x build.sh`
`./build.sh`

**Windows:**
`javac -d bin -cp "src" src/com/raycasting/*.java`
`java -Xmx2G -cp "bin" com.raycasting.Main`

**Sterowanie**
- WASD - ruch
- Mysz - obrót kamery
- Strzałki - alternatywny obrót
- SPACJA - zmiana trybu gry
- 1-4 - szybkie przełączanie trybów
- ESC - wyjście
